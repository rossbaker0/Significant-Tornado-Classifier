# ML Classification of Severe Thunderstorm Environments by Significant Tornado Occurrence

The Sounding Analog Retrieval System (SARS) is a commonly utilized system in operational meteorology. Specifically, it was designed to provide a variety of atmospheric sounding profiles from past severe weather episodes to use as reference analogs for forecasting future severe weather. 

Atmospheric soundings are vertical profiles of atmospheric conditions at a point, from the ground and up through the upper troposphere. The typical sources of sounding data are radiosondes, the instruments attached to weather balloons, and numerical weather prediction models. 

This repository documents my work towards the goal of synthesizing a skillful forecast model for detection of environments supporting significant tornadoes using atmospheric sounding profiles.
